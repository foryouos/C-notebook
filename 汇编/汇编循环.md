# 循环组成

* `循环体`  即	`loop指令`
* `循环次数`  即`CX寄存器` 的值

# loop

> `loop`中文：`循环`，下标的`标号`是我们在程序中`设置的标号`.
>
> `CPU`在执行`Loop`指令时，首先会先将`CX寄存器`中的`值减一`，然后`判断`CX的值`是否`为`零`，`不为零`的话则`跳转`到`标号处`继续`执行`，否则`继续`向`下`执行。

```assembly
loop 标号
```

# 执行顺序

```assembly
CX = CX - 1
@ 判断此时CX是否为0
@ 如果CX不为0，则转到标号处执行(jmp标号)
@ 如果CX为0，则继续向下执行
```

* 有点类似于do..while语句

```cpp
int cx;
// ...
do
{
    //执行循环逻辑
    // ...
    cx = xc -1; 
}while(cx > 0);
// ...
```

# CX寄存器

> `CX寄存器`的值相当于`循环`的`执行次数`，当循环次数为`0时`，也就是`CX`寄存器值为`0`，则`本次`循环`结束`



# 实际运用

```assembly
	mov cx,循环次数
s:
	循环执行的程序段
	loop s
```
